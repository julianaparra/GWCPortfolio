LED Light Show

To the first 30 seconds of the Stranger Things theme song, Caroline, Emma, Alaya, and I made a light show using 3 LEDs on a breadboard. The language we used is called C, through the program Arduino. It was pretty arduous because when we changed the timing of a light at any place in the song, the timing was off everwhere else so everything needed to be adjusted each time we changed anything.
