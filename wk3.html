Cybersecurity: Meshing HTML, CSS & JavaScript


This is a password evaluator! Using knowledge about privacy and security online as well as web development languages Hypertext Markup Language, Cascading Style Sheet, and JavaScript, I created a website that evaluates the strength of a password. It's based on whether or not the typed password is from the dictionary or not. As shown above, an alert pops up on the screen when a password is too simple (if it's just a word). Conversely, a congratulations screen appears when passwords are more complex.
